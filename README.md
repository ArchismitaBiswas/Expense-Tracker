# Expense Tracker

## Overview

A Python-based expense tracking system that records and analyzes personal expenses. The project stores expense data in CSV format, categorizes spending, generates reports, and visualizes spending patterns using charts.

## Features

* Expense recording and storage
* Category-wise expense analysis
* Monthly expense calculation
* Payment mode tracking
* Automated report generation
* Data visualization using charts

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* CSV

## Dataset Columns

* Date
* Category
* Amount
* Payment Mode
* Description

## Generated Outputs

* Expense Report
* Category-wise Expense Chart
* Monthly Expense Trend
* Payment Mode Distribution
